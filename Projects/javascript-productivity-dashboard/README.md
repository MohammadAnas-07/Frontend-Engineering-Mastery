# JavaScript Productivity Dashboard

A production-style JavaScript project built while mastering Frontend Engineering.

- ✅ Add Tasks
- ✅ Delete Tasks
- ✅ Complete / Incomplete Tasks
- ✅ Event Delegation
- ✅ Input Validation
